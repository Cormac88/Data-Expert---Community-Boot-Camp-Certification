# Data-Expert---Community-Boot-Camp-Certification
A free community boot camp that covers everything you need to know to go from junior data engineer to senior data engineer!

Topics that were covered in this boot camp:

1. Apache Kafka, Apache Flink, Apache Spark
2. Postgres
3. Fact Data Modeling
4. Airflow, Data Quality patterns
5. Data pipeline maintenance 
6. Data visualization with Tableau 
7. Data impact communication skills 
8. Analytical Patterns like Growth accounting and cumulative table design
